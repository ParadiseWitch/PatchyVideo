标题：
做了一个索引东方视频的网站，欢迎参观

内容：
网址：https://patchyvideo.com
问题：
1. 众所周知现在很多东方作者现在会将自己的视频上传到油管甚至推特，但是他们面临着传播途径的问题。如果你不刻意去搜索或者关注某个作者那么新人的作品将很难被人们发觉。
nico之类的网站可以提供标签直接查看新作品的功能（油管最多5个标签）。这让人联想到了各类图站，通过标签搜索图片，或许通过标签搜索视频也行得通。
2. 想必很多人都注意到了油管COPPA的事情，如果最坏的情况发生了，一个索引网站将会带来极大的便利。
3. 目前的已存在东方视频浩如烟海，但他们由于各种标题、简介、标签的问题，加之它们分散在各个视频网站上，使得它们无法被有效检索。我希望能有效整合这些视频。
4. 很多视频在nico/油管发布之后会被搬运工们搬运到b站上，有的甚至会被制作成中文内嵌/配音作品，相当于一个视频会在不同的网站上甚至在同一个网站上被发布多次。我希望能快速找到这些视频。
5. 很多剧情视频会构成一个系列，所以播放列表功能（playlist/mylist）这里显得尤为重要，但是b站的收藏夹不能方便的调整视频的顺序。
基于以上问题，patchyvideo诞生了。
网站名字是Patchouli's video library的简写，帕秋莉的视频图书馆。
核心功能：
1. 类似于各类图站的标签系统，用户可以自行创建标签，修改视频标签，根据标签查询视频。
2. 视频副本(copies)功能。用户可以添加/查看同一个视频在不同网站或者在同一个网站上的不同版本。
3. 播放列表(playlist)功能。
4. 网站目前正在开发中(包括多语言支持)，欢迎对未来的功能提出意见。
支持的网站:
1. b站
2. nico
3. 油管
4. 推特

欢迎加群757676234围观或者参与开发。目前需要前端（手机适配），后台（Python、C++），其他等。
如果你有什么意见或者问题，欢迎在下面指出。
